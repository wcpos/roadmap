# WCPOS Roadmap

Cross-repo planning and epics for the [WooCommerce POS](https://wcpos.com) project.

## How It Works

- **Epics** are parent issues here that link to implementation sub-issues in code repos
- **Proposals** are ideas being evaluated before committing to implementation
- Single-repo work stays in its own repo (`woocommerce-pos`, `monorepo`, `electron`, etc.)
